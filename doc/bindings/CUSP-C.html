CUSP for C++ quick start

Linux

Prerequisites

MLton compiler
  • Debian/Ubuntu: mlton (version 20100608 or better)
SQLite library
  • Debian/Ubuntu: sqlite3
  • Gentoo: dev-db/sqlite

Build the CUSP C bindings

cd <Bubblestorm root>/bindings/c/cusp
make
This will create libcusp.so.

Build the CUSP C++ bindings

cd <Bubblestorm root>/bindings/cpp/cusp
make
This will create libcuspcpp.a.

Build and run the CUSP C++ demos

cd <Bubblestorm root>/bindings/cpp/cusp/demo
make
This will create client and server. Run the server (listening at port 8585): 
./server
Run the client (using default port 8586 and connecting to localhost): 
./client

For running the demos, libcusp.so needs to be placed in the library search path. For testing purposes, you can manually override LD_LIBRARY_PATH, e.g., 

LD_LIBRARY_PATH=../../../c/cusp ./client
